Getting Started

How do I set up my admin account?

To set up your admin account:

  1. Log in with your provided credentials
  2. Complete your profile information in Settings
  3. Configure your notification preferences
  4. Set up your school information
  5. Add your first teachers and classes
What are the main features of the admin panel?

The admin panel includes:

  • Dashboard: Overview of school metrics and activities
  • Teacher Management: Add, edit, and manage teacher accounts
  • Class Management: Create and organize classes
  • Calendar: View and manage school events
  • Settings: Configure system preferences

Teacher Management

How do I add a new teacher?

To add a new teacher:

  1. Go to Teacher Management
  2. Click "Add Teacher" button
  3. Fill in the teacher's information (name, email, status)
  4. Assign classes if applicable
  5. Click "Add Teacher" to save
How do I assign classes to teachers?

You can assign classes in two ways:

  1. During teacher creation: Use the multi-select dropdown in the add teacher form
  2. After creation: Click the "Assign Classes" button next to any teacher in the list
What's the difference between Active, Inactive, and Archived status?
  • Active: Teacher can log in and use the system normally
  • Inactive: Teacher account is temporarily disabled
  • Archived: Teacher account is permanently removed from active lists but data is preserved

Class Management

How do I create a new class?

To create a new class:

  1. Navigate to Class Management
  2. Click "Add Class"
  3. Enter class details (name, grade level, description)
  4. Assign a primary teacher
  5. Set class capacity and schedule
  6. Save the class
Can I edit class information after creation?

Yes, you can edit class information at any time:

  1. Go to Class Management
  2. Find the class you want to edit
  3. Click the "Edit" button
  4. Make your changes
  5. Click "Save Changes"

Troubleshooting

What should I do if a teacher can't log in?

If a teacher can't log in:

  1. Check if their account status is "Active" in Teacher Management
  2. Verify their email address is correct
  3. Ask them to use the "Forgot Password" feature
  4. If issues persist, contact technical support
How do I reset a teacher's password?

To reset a teacher's password:

  1. Go to Teacher Management
  2. Find the teacher and click "Edit"
  3. Click "Reset Password" in the teacher details
  4. A temporary password will be emailed to the teacher
  5. The teacher will need to set a new password on first login
What should I do if the system is running slow?

If the system is running slow:

  1. Clear your browser cache and cookies
  2. Try using a different browser
  3. Check your internet connection
  4. Close unnecessary browser tabs
  5. If problems continue, contact technical support
